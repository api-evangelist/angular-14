# Angular 14 (angular-14)

Angular 14 is a major release of the Angular TypeScript framework from Google, released June 2, 2022. Key features include strictly typed reactive forms, standalone components (developer preview), protected component template members, extended template diagnostics, and improved CLI auto-completion. Angular 14 targets TypeScript 4.7 and ES2020, providing a comprehensive framework for building single-page applications.
